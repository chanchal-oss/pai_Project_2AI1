#  Logistic Regression Project

##  Objective

The objective of this project is to build a Machine Learning model using Logistic Regression to classify data into categories (0 or 1).

---

##  Dataset

* Dataset used: Titanic dataset
* Features: Age, Sex, Pclass
* Target: Survival (0 = No, 1 = Yes)

---

##  Steps Performed

1. Data preprocessing
2. Train-test split
3. Model training using Logistic Regression
4. Prediction on test data
5. Accuracy evaluation

---

##  Result

The model achieved high accuracy on the test dataset.

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn
